BLToolkit is provided as source code that you can use "as is" or customize for your applications. It is written in C# and compatible with .NET Framework 1.1 and 2.0.

Download one of the following zip-files. Unzip it. Add either BLToolkit.csproj (VS 2003) or BLToolkit.2.csproj (VS 2005) project into your solution. Add project references where you need it. Compile. Enjoy.
