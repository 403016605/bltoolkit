This example contains two demos to demonstrate the difference between ADO.NET and BLToolkit. The AdoDemo method represents a typical data access method which takes one parameter and returns a list of objects. All routine mapping work is done manually inside the method. The BLToolkitDemo method does the same work performed by BLToolkit.DbManager.
